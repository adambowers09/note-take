# note-take

## Description
An app for creating, saving, and editing and deleting text notes.

## Technologies Used
* Nodes.js
* Express.js
* Heroku CLI
* Javascript
* HTML
* CSS

## Installation
Server must have Node.js installed.  Run "npm i" frpm the root directory to install and run "npm start" to start the webserver.
